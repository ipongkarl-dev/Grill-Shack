# Grill Shack Restaurant Management App - PRD

## Architecture
- **Frontend**: React + Tailwind + Shadcn UI + Recharts (19 pages)
- **Backend**: FastAPI (30+ endpoints)
- **Database**: MongoDB

## All Implemented Features (19 Pages)

### Operations
1. Dashboard - KPIs, charts, stock alerts
2. Quick Mode - tap-to-count rapid entry
3. Session Input - full sales entry with real-time COGS
4. Daily Prep Checklist - auto-generated with checkboxes, print & CSV export
5. Alerts & Notifications - stock/cash/COGS alerts with severity levels

### Products
6. Products / COGS - product CRUD with dynamic cost calculations
7. Product Calculator - ingredient-level COGS breakdown
8. Margin Watch - profitability analysis with action recommendations

### Sales & Analytics
9. Sales Dashboard - trends, performance, CSV export
10. Weekly Control - week-by-week financial breakdown
11. Market Comparison - side-by-side ranking with radar chart

### Inventory & Planning
12. Stock Planner - demand forecasting
13. Inventory Tracker - stock movements with supplier info
14. Refill Cost Trends - supplier cost history tracking

### Financial
15. Cash System - reconciliation tracking
16. Allocation Tool - profit distribution
17. Cashflow Tracker - monthly targets vs actuals
18. Scale Planner - growth projection with capital requirements

### Configuration
19. Markets - full CRUD for market locations

## Test Results: 30/30 backend, 19/19 pages (100%)
